## 컴포넌트

각 컴포넌트는 모든 화면에서 같은 스타일을 유지함.

| 컴포넌트 | 내용 | 화면 | 위치 |
|------|-----------|-----------|-----------|
| 로고 | 서비스 로고 | s1, s2 | 화면 정중앙 |
| 로그인 버튼 | 디스코드 소셜 로그인 버튼 | s2 | 로고 바로 아레 |
| 성명 입력 필드 | 성명 입력 필드. 바로 아래에 '정산 시 사용되니 실명으로 입력해주세요.'라는 문구가 있음 | s3 | |
| 아이디 입력 필드 | 아이디 입력 필드. 우측에 중복 체크 버튼이 있음 | s3 | |
| 이메일 입력 필드 | 이메일 주소 입력 필드와, 메일 도메인 선택 드롭다운이 @를 사이에 둠 | s3 | |
| 가입하기 버튼 | 가입하기 버튼 | s3 | |
| 달력 | 달과 연도 선택 가능. 해당 달의 일정 표시. 선택된 날짜는 primary color, 나머지 일정 있는 날짜는 다른 색. '오늘' 일정이 있는데 선택되지 않았으면 회색 표시 | s4 | 콘텐츠 영역 상단 |
| 일정 | 달력에서 선택된 날짜와, 모임별 이벤트명 목록 | s4 | 콘텐츠 영역, 달력 바로 아래 |
| 검색바 | 이벤트 검색. 필터링(기간, 모임) 가능 | s4-1 | 바텀시트 최상단 |
| 이벤트 정보 카드 | 이벤트명, 모임명, 장소, 날짜 정보 | s4-1 | 바텀 시트 검색바 아래 |
| 모임 정보 카드 | 모임명 등 | s5 | 콘텐츠 영역 |
| 모임명 | 모임명 | s5-1, s5-2, s5-3 | 콘텐츠 영역 최상단, 상단바 바로 아래 |
| 탭 | 모임 페이지 내에서 공지, 이벤트, 멤버 탭 | s5-1, s5-2, s5-3 | 콘텐츠 영역, 모임명 바로 아래 |
| 공지 카드 | 공지 제목, 작성자 | s5-1 | 콘텐츠 영역 |
| 공지 작성 버튼 | 운영자에게만 보임. 플로팅 버튼 | s5-1 | 콘텐츠 영역 우측 하단 |
| 이벤트 카드 | 모임 내 이벤트 카드. 이벤트명, 날짜, 시간, 장소, 참여하기 버튼 | s5-2 | 콘텐츠 영역 |
| 이벤트 추가 버튼 | 운영자에게만 보임. 플로팅 버튼 | s5-2 | 콘텐츠 영역 우측 하단 |
| 탭 | 등록 QR 정보, 대시보드, 참가자 목록 탭 | s5-2-1-1, s5-2-1-2, s5-2-1-3 | 콘텐츠 영역 상단 |
| 사전 등록 링크 | 복사 가능 | s5-2-1-1 | 콘텐츠 영역, 탭 아래 |
| 입장 QR 코드 | 다운로드 가능 | s5-2-1-1 | 콘텐츠 영역, 사전 등록 링크 아래 |
| 입장 현황 | 사전 등록자 수, 실제 입장자 수, 현장 등록자 수, 참석 완료 비율(입장 인원/(사전등록+현장등록)) | s5-2-1-2 | 콘텐츠 영역 |
| 참가자 정보 카드 | 사전 등록된 인원 목록. 이름, 운영진/참가자 여부, 참여/대기/불참 상태 | s5-2-1-3 | 콘텐츠 영역 |
| 새 이벤트 기본 정보 | 새 이벤트 제목 입력 필드, 날짜, 시작 시간, 장소 | s5-2-2 | 콘텐츠 영역 |
| 상세 내용 | 이벤트 상세 내용 | s5-2-2 | 콘텐츠 영역 |
| 이벤트 생성하기 버튼 | 이벤트 생성하기 버튼 | s5-2-2 | 콘텐츠 영역 |
| 디스코드 채널 설정 토글 | 참가자 전용 비공개 채널 생성 여부 설정 | s5-2-2 | 콘텐츠 영역 |
| 참가자 정보 수집 카드 | 참가자 이름, 연락처 입력 | s5-2-2 | 콘텐츠 영역 |
| 멤버 정보 카드 | 멤버 이름, 운영진 여부, 가입일 | s5-3 | 콘텐츠 영역 |
| 이벤트 제목 | 이벤트 제목 | s5-2-3 | 콘텐츠 영역 |
| 이벤트 정보 | 일시, 장소, 참석 인원(사전 등록) | s5-2-3 | 콘텐츠 영역 |
| 상세 내용 | 이벤트 상세 내용 | s5-2-3 | 콘텐츠 영역 |
| 불참 버튼 | 이벤트 불참. 플로팅 버튼 | s5-2-3 | 콘텐츠 영역 하단 |
| 참여 버튼 | 클릭시 사전 등록 페이지로 이동. 설정된 시간 30분 전부터 '입장 QR 코드 찍기'로 변경. 플로팅 버튼 | s5-2-3 | 콘텐츠 영역 하단 |
| QR 체크인 | QR 찍기 | s5-4 | 콘텐츠 영역 |
| 참가자 직접 등록 | 사전 등록된 사람은 미리 등록된 정보 자동 이용, 현장 등록 시 참가자 정보(이름, 전화번호) 입력 뜸. 입장 처리 버튼, 취소 버튼 존재 | s5-4 | 콘텐츠 영역 |
| 프로필 카드 | 사용자 이름, 디스코드 아이디 등 정보 | s6 | 콘텐츠 영역 |
| 로그아웃 버튼 | 로그아웃 버튼 | s6 | 콘텐츠 영역 |
